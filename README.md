# Instagram Followers Exporter Chrome Extension

[🇺🇸 English](#english) | [🇪🇸 Español](#español)

## <a name="english">English Version</a>

### Overview
This Chrome extension allows you to export Instagram followers to a CSV or Excel file.

### Legal Warning and Meta Policy

#### 🚨 Important: Legal Risks and Limitations

##### Instagram/Meta Policies
- **Violation of Terms of Service**: Using unauthorized scraping or export tools may result in:
  - Temporary account suspension
  - Permanent account blocking
  - Legal actions by Meta

##### Ethical and Legal Considerations
- This tool is for **personal and educational use only**
- Do not use for:
  - Spam
  - Harassment
  - Mass data collection
  - Unauthorized commercial activities

##### Security Risks
- Meta can detect and block automated activities
- Possible API credentials revocation
- Risk of additional identity verification

### Disclaimer
⚠️ **The user assumes full responsibility for any misuse**

### Installation
1. Download extension files
2. Open Chrome and go to `chrome://extensions/`
3. Enable "Developer mode"
4. Click "Load unpacked" and select the extension folder

### Usage
1. Navigate to Instagram profile
2. Open followers list
3. Click extension icon
4. Select "Export Followers"
5. Download CSV or Excel file

### Notes
- Requires Instagram login
- Ensure full list loading
- Large lists may require manual scrolling

### Limitations
- Best performance with small lists
- Requires manual interaction with Instagram UI

### Best Practices
- Use in moderation
- Respect user privacy
- Do not share exported data

### Recommended Alternatives
- Use official Instagram API (if available)
- Request consent before collecting data
- Comply with data protection regulations (GDPR, CCPA)

### Support and Contributions
- Report issues on GitHub Issues
- Contributions improving legal compliance are welcome

## <a name="español">Versión en Español</a>

### Descripción General
Esta extensión de Chrome te permite exportar seguidores de Instagram a un archivo CSV o Excel.

### Advertencia Legal y Política de Meta

#### 🚨 Importante: Riesgos y Limitaciones Legales

##### Políticas de Instagram/Meta
- **Violación de Términos de Servicio**: El uso de herramientas de scraping o exportación no autorizadas puede resultar en:
  - Suspensión temporal de la cuenta
  - Bloqueo permanente de la cuenta
  - Acciones legales por parte de Meta

##### Consideraciones Éticas y Legales
- Esta herramienta es solo para **uso personal y educativo**
- No utilizar para:
  - Spam
  - Acoso
  - Recopilación masiva de datos
  - Actividades comerciales sin autorización

##### Riesgos de Seguridad
- Meta puede detectar y bloquear actividades automatizadas
- Posible revocación de credenciales de API
- Riesgo de verificación adicional de identidad

### Descargo de Responsabilidad
⚠️ **El usuario asume total responsabilidad por cualquier uso indebido**

### Instalación
1. Descargar archivos de extensión
2. Abrir Chrome en `chrome://extensions/`
3. Habilitar "Modo desarrollador"
4. Hacer clic en "Cargar extensión sin empaquetar"
5. Seleccionar carpeta de la extensión

### Uso
1. Navegar al perfil de Instagram
2. Abrir lista de seguidores
3. Hacer clic en ícono de extensión
4. Seleccionar "Exportar Seguidores"
5. Descargar archivo CSV o Excel

### Notas
- Requiere inicio de sesión en Instagram
- Asegurar carga completa de lista de seguidores
- Listas grandes pueden requerir desplazamiento manual

### Limitaciones
- Mejor rendimiento en listas pequeñas
- Requiere interacción manual con interfaz de Instagram

### Mejores Prácticas
- Usar con moderación
- Respetar privacidad de otros usuarios
- No compartir datos exportados

### Alternativas Recomendadas
- Usar API oficial de Instagram (si está disponible)
- Solicitar consentimiento antes de recopilar datos
- Cumplir con regulaciones de protección de datos (GDPR, CCPA)
